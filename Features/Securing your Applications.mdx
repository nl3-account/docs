Next Level3 Cloud Identity JIT Access allows an you to add your company applications to the portal, which will then require a Zero Trust-oriented MultiFactor Authentication to access them. Once an application is added, you can enable or disable account access to the application and monitor access activity. 

## Adding  an  Application  and  Domain  Validation 

> This process will require domain validation from an authoritative DNS server using a CNAME which will generate a sign in key that will be used to sign the JSON web tokens that are required to check lock statuses for users.
