Next Level3 Cloud Access JIT Identity Access allows an admininistrator to add their company applications to the portal and enables a Zero Trust-oriented MultiFactor Authentication platform. Once an application is added, the administrator can enable or disable account access to the application and monitor access activity. 

## Adding an Application and Domain Validation

> This process will require domain validation from an authoritative DNS server using a CNAME which will generate a sign in key that will be used to sign the JSON web tokens that are required to check lock statuses for users.
